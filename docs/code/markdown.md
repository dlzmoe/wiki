# markdown 语法

::: info
这是一条信息，info 后面的文字可修改
:::

::: tip 说明
这是一个提示，tip 后面的文字可修改
:::

::: warning 警告
这是一条警告，warning 后面的文字可修改
:::

::: danger 危险
这是一个危险警告，danger 后面的文字可修改
:::


::: details 点我查看
这是一条详情，details 后面的文字可修改
:::
